  • UpSlide.js

  • Give your new website life!

    With UpSlide.js you can create a unique website with a few lines of code!

  • Easily add more slides!

    All thumbnails are dynamically generated so you don't have to worry about the extra coding to create them!

  • Things you can do with it?

    • UpSlide.js works as a beautiful photo slider. But, I started this project having a more unique idea in mind.

    • Instead of having it scroll through pictures, why not have it scroll through pages of the site? You could obviously have it automatically scroll the pages but that isn't very practical (but it is definitely an option).

    • An Example of it being used as a website (other than this one) can be found here.

    • Subpages?

      Do you need to mimick subpages? Just nest another UpSlider inside the slide and initiate it!

  • How to use it?

    HTML...

    <div class="upSlide">
  <ul>	
    <li>
      <div style="display: block;background:url('img/slide1.jpg') no-repeat;background-position:center">
        <p>Slide 1 - This is the first slide!</p>
      </div>
    </li>

    <li>
      <div style="display: block;background:url('img/slide2.jpg') no-repeat;background-position:center">
        <p>Slide 2 - This is the second slide!</p>
      </div>
    </li>
  </ul>
</div>

    JavaScript...

    <script>
  $(document).ready(function(){
    $('.upSlide ul').upSlide();
  });
</script>
